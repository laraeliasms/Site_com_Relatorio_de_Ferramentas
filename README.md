# 💈 Site Barbearia --- Projeto de Desenvolvimento Web

## 📌 Objetivo

Desenvolvimento de um site institucional para uma barbearia, com foco em
presença digital, divulgação de serviços e facilidade de acesso às
informações pelos clientes.

## 🎯 Público-alvo

Homens jovens e adultos interessados em serviços de barbearia e estética
masculina.
